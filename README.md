## ■ 祈りの間（shrine.html）

神への祈りと献金を受け付けるインタラクティブな神殿である。
`shrine.html`をブラウザで開くと、祈りの間に入ることができる。

### 機能

**祈るボタン**

神に祈りを捧げる。神は常に見ている。押すたびに神からお告げが届く。


**献金ボタン**

金額を入力して献金を試みることができる。


**願いの入力欄**

自由に願いを文章で入力し、決定ボタンを押すと神々しい音楽が流れ、神からの応答が届く。

**C言語コンパイラ**

願いの入力欄に`#include`や`int main`を含むCのコードを入力すると、コンパイルボタンが出現する。
神は[Wandbox](https://wandbox.org)を通じてコードをコンパイル・実行し、結果を返す。

```c
#include <stdio.h>
int main() {
    printf("Hello, God\n");
    return 0;
}
// → 神のコンパイル結果： Hello, God
```

### 使い方

```bash
# cloneしてブラウザで開くだけでよい
git clone https://github.com/KanekoMasayoshi/kami.git
cd kami
open shrine.html      # macOS
# または shrine.html をダブルクリック（Windows / Linux）
```

> ⚠️ C言語コンパイル機能はインターネット接続が必要です（Wandbox APIを使用）。
